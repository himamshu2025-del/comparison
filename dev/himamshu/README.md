# TradeFlow Dashboard - Sustainable Trade & UN Goal Focus

This interactive dashboard by **Himamshu Aluru** for **ModelEarth** provides insights into sustainable trade practices worldwide, leveraging real-world data from **UNCTAD**, **UNEP**, **UNIDO**, **OECD**, and other UN sources. It focuses on **trade flows of biodiversity-based products, green goods, and sustainability standards** per country.

## Features

- **Generate 10 Global Insights** with a click of a button.
- **Select a country** to see 10 country-specific insights.
- **Interactive charts** showing biodiversity-based export values.
- **Country icons and images** for visual guidance.
- **Polished UI** with responsive cards, buttons, and dropdown selectors.

## Countries Included

Germany, USA, China, India, Netherlands, France, Norway, Switzerland, UK, Brazil
